Associated Program and Data

This is the home page of our obtained transposone exposed spontaneous mutation rate constants. The data, at present, are for the human nuclear genome. To calculate the full set of spontaneous rate constants for any human nucelar DNA sequence, you can either download the stand-alone multicore version of the Trek mapper program (left), or use our web server implementation accessible through HERE (WEB SERVER). The latter is only applicable for relatively short sequences, to conform the available server resources. The source data and the full parameter sets for the human genome are deposited in the Supplementary Information of the publication. The manuscript is currently in press. The preprint, along with the source data, are available from bioRxiv: 

Aleksandr B. Sahakyan and Shankar Balasubramanian, “Core variability in mutation rates shapes the basal sequence characteristics of the human genome”, bioRxiv, http://dx.doi.org/10.1101/024257
